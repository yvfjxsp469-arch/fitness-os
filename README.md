# Fitness OS

Personal fitness management system built with Next.js, TypeScript, Prisma and PostgreSQL.

## Features

- Weight Tracking
- Nutrition Tracking
- Training Tracking
- Analytics Dashboard
- Personal Record (PR) Tracking

## Tech Stack

- Next.js 16
- TypeScript
- Prisma
- PostgreSQL (Supabase)
- TailwindCSS
- shadcn/ui
- Recharts
